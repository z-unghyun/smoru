# SMORU

SMORU is a native iOS wellness app that adapts a morning routine based on sleep condition.

## Required output

The deliverable must be a Swift + SwiftUI iOS app, not a web prototype.

Required:
- Xcode iOS project
- SwiftUI UI
- HealthKit authorization and data fetching
- SwiftData local persistence
- UserNotifications local notifications
- Sign in with Apple and Google Sign-In integration structure
- Firebase placeholder configuration without real credentials

Not accepted:
- single HTML file
- React, Vite, or Next.js web app
- static mockup only
- hardcoded API keys

Read first:
1. docs/AGENT_IMPLEMENTATION_SPEC.md
2. docs/smoru_prd_v1_native_addendum.md
3. docs/smoru_srd_v1_native_addendum.md

If documents conflict, the Agent Implementation Spec and V1 Native Addendum override older drafts.
